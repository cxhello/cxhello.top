## Nexus Maven Repository Deploy

### 持久化目录增加权限

```bash
chmod 777 nexus-data
```

### Docker Compose

```bash
version: '3'
services:
  nexus3:
    image: sonatype/nexus3
    container_name: nexus3
    ports:
      - "8081:8081"
    volumes:
      - "/etc/localtime:/etc/localtime"
      - "./nexus-data:/nexus-data"
    restart: unless-stopped
```
